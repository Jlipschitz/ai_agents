# Example Repos

These directories are copyable starting points for using `ai-agents` in common repo shapes.

Each example includes:

- `agent-coordination.config.json`
- A small `package.json` with coordination scripts
- `docs/ai-agent-app-notes.md`
- A short README with the intended workflow

Use an example by copying its files into a new repository, installing this package or copying the coordinator scripts, then running:

```bash
npm run agents:doctor
npm run agents:plan
```

## Available Examples

- `basic-node`: a small Node service or library.
- `react-app`: a frontend app with visual-check hooks.
- `docs-only`: a documentation repository with docs-focused planning.
